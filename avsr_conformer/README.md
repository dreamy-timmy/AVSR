# AVSR Conformer

Алгоритм автоматического распознавания русскоязычной речи с гибридным 
декодером на основе свёрточно-трансформерного энкодера.

## Описание

Реализация системы распознавания речи (ASR), основанная 
на архитектуре Conformer с гибридным CTC/Attention декодером и 
опциональной аудио-визуальной веткой для повышения устойчивости 
к акустическому шуму.

## Архитектура

- **Энкодер**: Conformer (6 блоков, d_model=256, 4 головы внимания)
- **Декодеры**: гибридный CTC + Attention
- **Видеоветка**: CNN-энкодер с кросс-атентным слиянием
- **Параметров**: 16.7M
